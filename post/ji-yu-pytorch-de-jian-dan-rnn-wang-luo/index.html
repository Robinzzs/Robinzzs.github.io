Pytorch学习——简单RNN网络

· 2020-06-02 ·

Preknowledge

考虑一个预测是否下雨的模型，每一天都有若干个特征的数据，需要若干天的数据来进行预测。
假设有如下数据记录，并且利用前三天的数据来预测后一天。

day 温度 气压 雨
1 26 960 True
... ... ... ...

模型的输入包含三种特征：
温度x1={t1,t2,t3}x_1=\{t_1,t_2,t_3\}x1​={t1​,t2​,t3​}
气压x2={p1,p2,p3}x_2=\{p_1,p_2,p_3\}x2​={p1​,p2​,p3​}
是否下雨x3={r1,r2,r3}x_3=\{r_1,r_2,r_3\}x3​={r1​,r2​,r3​}

若使用全连接神经网络（CNN）可以将数据组合成9维的长向量X={x1,x2,x3}X=\{x_1,x_2,x_3\}X={x1​,x2​,x3​}，然后将若干天的数据进行整理，每四天作为一组，其中三天组合成长向量XXX，第四天作为预测（输出）的标签。

由于全连接网络（稠密网络Dense/Deep Network）中，权重项是很多的，处理序列式的数据类型（表格、语音、视频等）对网络的训练来说是非常大的挑战。

  • 卷积层（权重共享，“降维”）
    若input通道为128，output为64，卷积核conv为5*5，权重项为25×27+625\times2^{7+6}25×27+6，需要计算20W个权重项。
  • 全连接层（通道数较多）
    若input通道为4096，output为1024，权重项4096×10244096\times10244096×1024，需要计算420W个权重项。

利用RNN处理带有序列模式的数据（天气、股市、自然语言等），并利用权重共享的概念来减少需要训练的权重项的数量。

  • 考虑数据块与权重的连接关系
  • 考虑数据块本身的前后时间序列关系

RNN cell函数

基本的RNN模型实现

Property

  • RNN cell的本质是一个线性层（linear layer），将一个维度空间映射到另外一个维度空间中去。
    xt→RNNcell→htx_t \rightarrow RNNcell \rightarrow h_txt​→RNNcell→ht​
    若输入层（input layer）的xtx_txt​为三维数据，隐藏层（hidden layer）的hth_tht​为五维数据，

  • RNN cell的性质是权重共享的，在一个时间序列上的各个时间点上的数据共享相同的RNN层。
    ht=tanh⁡[(xtwih+bih)+(ht−1whh+bhh)]h_t = \tanh [(x_{t}w_{ih} +b_{ih}) + (h_{t-1}w_{hh}+b_{hh})]ht​=tanh[(xt​wih​+bih​)+(ht−1​whh​+bhh​)]
    其中
    ht−1∈Rhidden_sizeh_{t-1} \in R^{hidden\_size}ht−1​∈Rhidden_size
    xt∈Rinput_sizex_{t} \in R^{input\_size}xt​∈Rinput_size
    权重wihw_{ih}wih​的大小为hidden_size×input_sizehidden\_size \times input\_sizehidden_size×input_size
    权重whhw_{hh}whh​的大小为hidden_size×hidden_sizehidden\_size \times hidden\_sizehidden_size×hidden_size
    两权重参与到每一个时间点上input数据的运算，即权重共享。

Data size

# 定义
cell = torch.nn.RNNCell(input_size=input_size, hidden_size=hidden_size)
# 循环调用
hidden = cell(inputs, hidden)

inputs大小(batch_size,input_size)
hidden大小(batch_size,hidden_size)

  • 假设数据序列有如下的性质
    批次大小batch_size，
    序列长度seq_len，
    输入大小input_size，
    隐层大小hidden_size

  • RNN cell的输入输出大小为
    input.shape = (batch_size, input_size)
    output.shape = (batch_size)

  • 数据张量的维度为
    dataset.shape = (seq_len, batch_size, input_size)

范例

import torch

batch_size = 1
seq_len = 3
input_size = 4
hidden_size = 2

cell = torch.nn.RNNCell(input_size=input_size, hidden_size=hidden_size)

dataset = torch.randn(seq_len, batch_size, input_size)
hidden = torch.zeros(batch_size, hidden_size)

for idx, input in enumerate(dataset):
    print("=" * 20, idx, "=" * 20)
    print("Input size:", input.shape)

    hidden = cell(input, hidden)

    print("Output size:", hidden.shape)
    print(hidden)

RNN函数

完整的RNN模型及过程实现，包括其中的数据循环过程。

Date size

# 定义
cell = torch.nn.RNN(input_size=input_size, hidden_size=hidden_size, num_layers=num_layers)
# 调用
out, hidden = cell (inputs, hidden)

调用RNN不用写循环，函数自动计算

  • 假设数据序列有如下的性质
    批次大小batch_size，
    序列长度seq_len，
    输入大小input_size，
    隐层大小hidden_size
    隐层数量num_layers

  • 输入数据维度
    inputs.shape = (seq_len, batch_size, input_size)
    hidden.shape = (num_layers, batch_size, hidden_size)

  • 输出数据维度
    out.shape = (seq_len, batch_size, hidden_size)
    hidden.shape = (num_layers, batch_size, hidden_size)

  • batch_first参数，如果设置为True，则输入输出张量结构为（batch_size, seq_len, input_size）

范例

import torch

batch_size = 1
seq_len = 3
input_size = 4
hidden_size = 2
num_layers = 1

cell = torch.nn.RNN(
    input_size=input_size, hidden_size=hidden_size, num_layers=num_layers
)

#
inputs = torch.randn(seq_len, batch_size, input_size)
hidden = torch.zeros(num_layers, batch_size, hidden_size)

out, hidden = cell(inputs, hidden)

print("Output size:", out.shape)
print("Output:", out)
print("Hidden size:", hidden.shape)
print("Hidden:", hidden)

RNN模型的一些问题

  • 梯度消失和梯度爆炸问题
    根据反向传播（链式求导法则），梯度会不断相乘，很容易造成梯度消失或者梯度爆炸等情况

  • loss数值降不下来
    一个hidden_size=12的单点数据预测的例子，optimizer为Adam，loss function为MSELoss，可以发现Loss值在并没有一直单调递减。
